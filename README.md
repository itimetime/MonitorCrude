## 纽约原油（美原油）监控助手

#### 使用说明：

1.安装python

2.在命令窗口输入`pip install request`

3.在命令窗口 cd 进项目文件夹

4.打开config.py修改相关配置（邮箱刷新频率等）

5.命令窗口输入命令 `python Mainactivity.py` 

#### 功能说明：

1.设置低价、高价提醒

2.邮件提醒

3.设置多个价格，价格触发后会发送多封邮件，把握价格走势

4.默认为美元（数据来自新浪财经），支持汇率设置
